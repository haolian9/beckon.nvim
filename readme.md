a fuzzy matching picker


## motivation
i need a lightweight fuzzy matching picker in nvim, to replace fzf in some usecases:
* buffers, arglist
* ui.select
* tags
* lsp definitions
* ...


## status

* far from usable

* haolian9/infra.nvim
